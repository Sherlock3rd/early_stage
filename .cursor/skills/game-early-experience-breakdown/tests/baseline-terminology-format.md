# 飞书术语与格式修订基线

## 真实失败表现

当前 `寒霜拆解` 页签暴露出以下问题：

- 任务链写成逐步操作流程，而不是该时间片的主要任务。
- 核心循环写成玩家操作链，而不是系统正在教学的玩法循环。
- 渐进体验、地图体验和经济体验混入评价与推断，未分别聚焦系统解锁、地图解锁和成长数值。
- 剧情轴重复输出“事实/推断/高潮/低谷/心流/证据”，没有直接说明剧情内容及叙事目的。
- 无信息单元格仍填写“未观察到”和通用推断，横向阅读噪声过高。
- 行标签缺少参考页签使用的中英术语，如“渐进体验预期 (New Content)”。

## 修订验收

- 七维显示标签与 `ai学习` 页签一致。
- 单元格只保留用于横向比较的结论，不展示“事实：”“推断：”前缀。
- 无信息允许空置；不得为了填满表格补写通用句。
- 剧情轴只写“发生了什么”与“叙事目的”，不显示高潮/低谷/心流为“无”的说明。
