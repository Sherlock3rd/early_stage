# 游戏前期体验拆解：情绪压力与整切线交互设计

## 背景

现有双曲线已经将体验整体趋势改为线性回归，但仍有三个问题：

1. 桌面绘图区最大高度仅 420px，高分节点、剧情高潮菱形和心流星标的辨识空间不足。
2. 曲线只能在节点附近悬停和点击，用户难以沿同一时间切线快速预览和跳转。
3. 情绪强度被限定为剧情刺激，遗漏暴风雪倒计时、严寒威胁、资源告急等游戏主动制造的环境压力，
   导致大量时间片错误地显示为 0。

## 目标

- 扩大桌面绘图区，并保证所有标记处于安全边界内。
- 让每个时间片在完整绘图区高度上都可悬停、聚焦和点击。
- 将情绪曲线从“纯剧情强度”修正为“以剧情为绝对主体的综合情绪唤起强度”。
- 通过结构化子分和刺激来源，防止后续分析再次只看剧情或无证据补分。
- 重新检查寒霜全部 28 个时间片，并用录屏证据重评情绪数据。

## 非目标

- 不新增独立环境压力曲线。
- 不修改体验强度、体验线性回归算法或七维飞书内容。
- 不修改飞书页签。
- 不用固定模板把所有 0 分机械改为非 0；没有可观察刺激时仍允许为 0。

## 情绪评分模型

### 子分

每个时间片分别记录：

- `narrative_score`：剧情刺激强度，范围 `0～5`。
- `supporting_score`：其他情绪刺激强度，范围 `0～5`。

其他刺激包括环境压力、紧迫目标、战斗冲突、成长奖励和危机缓解。最终情绪强度为：

```text
intensity = round(
  0.7 × narrative_score
  + 0.3 × max(narrative_score, supporting_score),
  1
)
```

该公式保证：

- 当其他刺激不高于剧情刺激时，最终强度等于剧情分，剧情保持绝对主体。
- 剧情 5 分时最终仍为 5 分，不会被 70% 权重压低。
- 纯环境或系统刺激最高贡献 1.5 分，能够显示持续压力，但不能取代剧情高潮。
- 其他刺激高于剧情刺激时，只对最终强度提供最多 30% 的抬升。

### 剧情子分

- `0`：无剧情刺激。
- `1`：轻微氛围、关系或信息铺垫。
- `2`：明确剧情推进，但刺激有限。
- `3`：冲突、胜利、失落或重要揭示。
- `4`：强烈转折或关键角色事件。
- `5`：核心剧情高潮或全段最强剧情事件。

### 其他刺激子分

- `0`：没有可观察的其他情绪刺激。
- `1`：背景性氛围、轻度期待或轻微缓解。
- `2`：明确警告、倒计时、资源压力、奖励或安全感变化。
- `3`：持续影响玩家目标与决策的危机、紧迫感、战斗或成长反馈。
- `4`：立即危机、高压战斗、重大成长奖励或显著危机解除。
- `5`：全段最强的非剧情刺激。

### 刺激来源

`drivers` 是可多选数组，只允许：

- `narrative`
- `environment_pressure`
- `urgency`
- `combat`
- `progression_reward`
- `relief`

规则：

- `narrative_score > 0` 时必须包含 `narrative`。
- `supporting_score > 0` 时必须包含至少一个非 `narrative` 来源。
- 两项子分均为 0 时，`drivers` 必须为空。
- `intensity > 0` 时，`event` 与 `reason` 必须是非空文本。
- `event` 描述当前时间片可观察到的主要情绪刺激，不再只描述剧情事件。
- `valence` 继续使用 `positive`、`negative`、`mixed`、`neutral`，表示整体情绪方向。

### 数据示例

```json
{
  "emotion": {
    "narrative_score": 1,
    "supporting_score": 3,
    "intensity": 1.6,
    "valence": "negative",
    "drivers": ["narrative", "environment_pressure", "urgency"],
    "event": "暴风雪倒计时持续逼近，幸存者需要尽快恢复供暖",
    "reason": "轻度剧情推进叠加持续环境威胁，形成明确紧迫感"
  }
}
```

## 寒霜重评

- 重评全部 28 个时间片，不只修改现有 0 分片。
- 先检查主图；涉及倒计时变化、字幕、音频、危机触发或缓解时查看必要原视频片段。
- 重点核对：
  - 暴风雪还有多久到来及倒计时变化；
  - 严寒、供暖、资源或幸存者安全压力；
  - 战斗前后的威胁与缓解；
  - 建筑升级、系统解锁和成长奖励；
  - 比尔去世、送别和继承遗志等关键剧情。
- 无法从画面、字幕、音频或连续上下文确认时，不增加子分，并记录待确认。
- 比尔去世继续是剧情主峰；环境压力只能作为辅助来源，不得把普通倒计时抬到同级。

## 响应式绘图区

- 桌面端 `#emotion-curve-svg` 高度使用 `clamp(480px, 42vw, 600px)`。
- 移动端保持 `320～420px`，避免曲线区占据过多纵向空间。
- SVG 继续使用容器实际像素宽高作为 ViewBox，并通过 `ResizeObserver` 重绘。
- 绘图区顶部预留足够标记安全区；菱形、星形、圆点和文本不得越过 SVG 边界。
- 页面主网格继续使用 `minmax(0, 1fr)`，禁止图框横向溢出。

## 全高时间片命中带

### 几何

每个时间片生成一个透明矩形命中带：

- 高度覆盖整个有效绘图区。
- 第一个命中带从绘图区左边界开始。
- 相邻命中带边界取两个时间点横坐标的中线。
- 最后一个命中带延伸到绘图区右边界。
- 命中带必须连续、无空隙、无重叠，并与时间片索引一一对应。

### 交互

- 命中带内任意纵向位置悬停，都显示该时间片 Tooltip。
- 点击命中带任意位置，都联动下方对应时间片详情。
- 命中带支持 `tabindex=0`、Enter 和 Space。
- 筛选隐藏的时间片命中带降低透明度、设置 `aria-disabled=true`，并禁止跳转。
- 当前选中时间片命中带设置 `selected` 和 `aria-current=true`。
- 悬停或聚焦时显示淡色竖向高亮带与中心引导线。
- 原曲线圆点只负责视觉表达，不再承担主要命中区域。

### Tooltip

Tooltip 增加：

- 最终情绪强度；
- 剧情子分；
- 其他刺激子分；
- 刺激来源标签；
- 主要刺激事件与评分原因；
- 原有体验分、四项体验依据和回归预测值。

## 校验

`analysis_model.py` 必须：

- 要求 `narrative_score`、`supporting_score`、`intensity`、`valence`、`drivers`、`event`、`reason`。
- 校验两个子分均为 `0～5` 的有限数。
- 按规定公式复算 `intensity`，要求精确到一位小数并一致。
- 校验 `drivers` 是无重复的规定枚举数组。
- 校验子分与来源之间的一致性。
- 保留情绪方向、事件、原因、时间片对齐及体验依据的既有校验。
- 拒绝缺少新子分或来源字段的旧情绪点。

## 数据流

1. 分析者查看寒霜全部时间片与必要原视频。
2. 为每片填写剧情子分、其他刺激子分、来源、方向、事件与原因。
3. 校验器复算并验证最终情绪强度。
4. Viewer 原样读取子分与来源，负责绘图、Tooltip 和交互。
5. 构建器重建本地 `viewer/index.html` 与 `data.json`。
6. 飞书输出继续只读取既有七维 `fact`，不受本轮影响。

## 错误处理

- 子分越界、公式不一致、来源缺失或非法时停止构建。
- 无可靠证据时保留 0 分或待确认，不用模板补分。
- 图表尺寸或命中带渲染失败时显示可见错误，但不阻断时间片详情。
- ResizeObserver 重绘后必须恢复选中、筛选和 ARIA 状态。

## 测试与验收

自动化测试覆盖：

- 70/30 主体公式、四舍五入、剧情 5 分保持 5、纯其他刺激最高 1.5。
- 子分范围、公式一致性、drivers 枚举、去重和条件必填。
- 寒霜 28 片全部符合新契约。
- 桌面端高度 480～600px，移动端高度 320～420px。
- 标记安全边界。
- 28 条命中带连续覆盖绘图区，无空隙和重叠。
- 命中带顶部、中部和底部均返回同一时间片。
- 悬停、点击、键盘、筛选禁用、Tooltip、ResizeObserver 状态保持。
- 既有曲线开关、线性回归、时间轴、七维详情和图片轮播无回归。

浏览器验收至少覆盖 1440px 桌面端和 390px 移动端：

- 图框完整显示，标记不裁切，无横向滚动。
- 任意时间片命中带的顶部、中部和底部均可预览与点击。
- Tooltip 展示两项子分和刺激来源。
- 比尔去世仍是主要剧情峰值。
- 暴风雪倒计时等环境压力在有证据的时间片形成非零情绪值。
