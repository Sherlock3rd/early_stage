# Early Stage Experience Breakdown

游戏前期体验拆解的独立备份与静态发布仓库。

## GitHub Pages

- 查看器：<https://sherlock3rd.github.io/early_stage/>
- 分析数据：<https://sherlock3rd.github.io/early_stage/data.json>

仓库根目录固定保留 `index.html`、`data.json` 和 `screenshots/`，因此 GitHub Pages 可直接加载数据与截图。

## 内容

- `.cursor/skills/game-early-experience-breakdown/`：可复用 Skill、脚本和测试。
- `docs/superpowers/`：设计规格与实施计划。
- `session/requirements/`：需求记录。
- `mistakes/`：复盘规则。
- 根目录静态文件：当前寒霜前期体验拆解页面。
